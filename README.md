<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
In this Lab, we will guide you step-by-step in creating an Active Directory home lab environment using Oracle VirtualBox. By configuring and running this lab, you will enhance your comprehension of Active Directory and Windows networking.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
